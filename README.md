# Roadmap Belajar Python: Project Latihan

Selamat datang di repositori latihan Python. File ini dirancang untuk memberikan panduan langkah-demi-langkah dalam mempelajari dasar-dasar pemrograman Python menggunakan materi yang tersedia di folder `project_latihan`.

---

## 🚀 Urutan Pembelajaran

Ikuti urutan di bawah ini untuk membangun fondasi yang kuat:

### 1. Dasar-Dasar & Tipe Data (`soal1`)
Fokus pada cara Python menyimpan data dan operasi matematika dasar.
*   **Materi:** `Belajar_tipe_data.py`, `boolean.py`
*   **Praktik:** `kalkulator_sederhana.py`
*   **Studi Kasus:** `soal_login.py` (Logika dasar)

### 2. Logika Pemrograman & Flow Control (`soal2` & `soal3`)
Belajar bagaimana program mengambil keputusan dan melakukan tugas berulang.
*   **Kondisional:** `pertanyaan.py`, `login.py`
*   **Perulangan (Loops):** `whileandloop.py`
*   **Logika Visual:** `membuat_bintang.py` (Latihan nested loop)
*   **Evaluasi:** `ujian_2.py`

### 3. Fungsi / Function (`soal4`)
Mengorganisir kode agar lebih efisien dan dapat digunakan kembali.
*   **Materi Utama:** `function.py`
*   **Pengayaan:** `ujian_tambahan.py`
*   **Integrasi:** `ujian_1-4.py`

### 4. Struktur Data Dinamis (`soal5`)
Mempelajari cara mengelola kumpulan data yang lebih kompleks.
*   **List:** `List.py`, `looplist.py`, `list_dan_pertambahan.py`
*   **Dictionary:** `dictionary.py`
*   **Manipulasi Data:** `filter.py`, `tambah.py`
*   **Latihan Umum:** `Latihan_dasar.py`

---

## 🛠️ Cara Menjalankan Program

Pastikan Anda telah menginstal [Python](https://www.python.org/). Jalankan file melalui terminal dengan perintah:

```bash
python project_latihan/soalX/nama_file.py
```
